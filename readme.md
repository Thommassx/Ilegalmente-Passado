# EcoFactory

## Sistema de Monitoramento e Gestão de uma Indústria Inteligente

### Descrição

O EcoFactory é uma aplicação Web Full Stack desenvolvida para centralizar o gerenciamento de uma indústria inteligente.

O sistema permite cadastrar máquinas, registrar produções, acompanhar indicadores de produtividade e visualizar informações importantes por meio de um dashboard interativo.

O objetivo é substituir controles realizados em planilhas por uma plataforma moderna, segura e integrada.

# Como executar

## Clonar o projeto


```bash

```

## Back-End

```bash
cd backend
npm install
npm start
```
